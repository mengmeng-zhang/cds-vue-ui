# cds-vue-ui
> 基于elementUI的二次封装组件
> 内容在不断完善中

#### 安装
```bash
npm install --save cds-vue-ui
```

#### 使用
``` javascritp
// 全局引入
import Vue from 'vue'
import Cds from 'cds-vue-ui'

Vue.use(Cds)

// 单个包引入
import { Icon } from 'cds-vue-ui'

{
    components: {
        Icon
    }
}
```
